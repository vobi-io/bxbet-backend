# BX.BET README FILE

#### Deployment to Ropsten network
``` truffle migrate --network ropsten ```

#### Truffle Install
``` npm install -g truffle ```

#### Compile Contracts
``` truffle compile ```

-----------

```shell
node_modules/.bin/ganache-cli

truffle exec ./contracts/seed.js  run .js file by truffle
```
